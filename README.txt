Hello, to run our application, clone the repo in Xcode. Then open a terminal window on your machine:
1. Install homebrew if you don't already have it. Follow instructions at https://brew.sh/
2. Install Cocoapods. instructions at https://formulae.brew.sh/formula/cocoapods
3. Navigate to the directory with the cloned repo, and on the same level as the Podfile run 'pod install'. NOTE: you may need to delete 'Podfile.lock', 'Derived Data', and the 'Pods' folder and try 'pod install' again if you get an error
4. open the UWTickets+.xcworkspace file to run the app.
